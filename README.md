# 📊 Reel Counter Chrome Extension

Track how many reels/shorts you watch daily on Instagram and YouTube — and take control of your scrolling habits.

---

## 🚀 Features

- 📊 Real-time reel counter
- 📅 Automatic daily reset
- 📈 Weekly usage graph (visual stats)
- ⚠️ Custom usage limit alerts
- 🎛️ Clean floating UI with settings panel
- 🧲 Smooth draggable widget

---

## 🎯 Why This Project?

Most people don’t realize how much time they spend scrolling.

This extension helps you:
- Become aware of your usage
- Build better habits
- Reduce time wasted on short-form content

---

## 📦 Installation

1. Download or clone this repository  
2. Open Chrome and go to: chrome://extensions/
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked**
5. Select this project folder
